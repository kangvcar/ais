## 版本说明

### 版本格式
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 更新类型
- `新增` - 新功能
- `改进` - 现有功能的改进
- `修复` - 问题修复
- `删除` - 已移除的功能
- `安全` - 安全相关的修复

### 获取最新版本
```bash
# 升级到最新版本
pip install --upgrade ais-terminal

# 查看当前版本
ais --version
```
